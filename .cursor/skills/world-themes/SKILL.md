# World themes

Read [world-sheets](../world-sheets/SKILL.md) first, then this skill.

## Goal

Name a recurring motif and show how it surfaces — for readers and for staying intentional while writing.

Seed: `content/themes/memory-in-the-water.md`.

## Steps

1. Read `content/themes/_template.md` and existing theme sheets.
2. One motif per file; title should be evocative but clear (`memory-in-the-water`).
3. Ask:
   - Where has this already appeared (chapters or other sheets)?
   - What must it *not* turn into (cliché to avoid)?
4. Fill **Motif**, **How it appears**, **Notes**.
5. Tie **How it appears** to concrete images (water, light, speech, absence) not abstract essays.
6. Themes guide tone; they should not spoil plot machinery.

## Style

- Short and sharp. Prefer image lists and chapter pointers over theory.
- Leave room for the motif to evolve — mark provisional readings in Notes.

## Done when

- Valid frontmatter (`name`, `order`, `draft`)
- Motif stated in one or two sentences
- At least one concrete appearance note (even if “planned / not yet on page”)
