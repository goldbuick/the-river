# World timeline

Read [world-sheets](../world-sheets/SKILL.md) first, then this skill.

## Goal

One sheet per known event or era marker. Show what readers may treat as chronology without spoiling unpublished twists.

Seed: `content/timeline/the-river-arrives.md`.

## Steps

1. Read `content/timeline/_template.md` and existing timeline sheets.
2. Sort with `order` (chronological). Use `when` for a freeform label (“Before the story opens”, “Day one”, etc.).
3. Ask:
   - Is this established on-page, implied, or still provisional?
   - What must stay unknown?
4. Fill **What happened**, **What is known**, **Notes**.
5. Prefer short entries; split large arcs into multiple sheets rather than one essay.
6. If chapters contradict an entry, update the sheet or flag the conflict to the user.

## Style

- Past tense for settled events; hedge clearly when provisional (“said to…”, “not yet shown”).
- No fake calendar precision unless the story has one.

## Done when

- Valid frontmatter (`name`, `order`, `draft`; optional `when`)
- Entry ordered relative to siblings
- Distinguishes fact vs. rumor when both appear
