# World glossary

Read [world-sheets](../world-sheets/SKILL.md) first, then this skill.

## Goal

Short, stable definitions for terms and lore crumbs worth tracking — not a full encyclopedia.

Seed: `content/glossary/the-banks.md`.

## Steps

1. Read `content/glossary/_template.md` and existing entries.
2. One term per file; slug matches the term when practical (`the-banks.md`).
3. Set optional `category` (`place-lore`, `object`, `term`, …).
4. Ask only if needed: in-story meaning vs. ordinary meaning; any taboo or double sense?
5. Write a tight **Definition**; put speculation under **Notes**.
6. Link related locations/characters by name in prose when helpful — no special syntax required yet.

## Style

- Dictionary-brief. One idea per entry.
- Prefer what a careful reader can verify from published (or soon-public) text.

## Done when

- Valid frontmatter (`name`, `order`, `draft`; optional `category`)
- Definition stands alone as an index card
- No unused jargon invented “for flavor” without a story hook
